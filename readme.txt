== Description ==
Awesome Login Customizer is a WordPress plugin that allows you to easily customize the default login page. Give your login page a personalized touch to match your site's branding and improve user experience.


*   [Need Help?](https://github.com/ripon181)		
*   [About Author](https://github.com/ripon181)

==  Key Features ==
*   Change WordPress Default logo.  
*   Change Backdround Image
*   Change login button color

= Usage =

* Go to your Dashboard after installation and navigate to Awesome Login unser settings to configure the accordion.


== Installation ==

* Upload the `awesome-login-customizer` folder to the `/wp-content/plugins/` directory.
*Activate the plugin through the 'Plugins' menu in WordPress.
*Go to the 'Awesome Login' settings in the WordPress admin dashboard to start customizing your login page.


= Minimum Requirements =

* WordPress 4.5 or greater
* PHP version 5.6 or greater
* MySQL version 5.0 or greater


= UPDATING =

Automatic updates should work seamlessly. We always suggest you backup your website before any automated update to avoid unforeseen problems.

== Frequently asked questions ==
= How do I upload a custom logo? =
Navigate to the 'Login Customizer' settings, and you'll find an option to upload your custom logo.

= Can I revert to the default login page settings? =
Yes, at any time, you can deactivate the plugin to revert to the default WordPress login page.
